# Ant Colony Optimization Algorithm (ACO)

Implementación en **Haskell** de una colonia de hormigas para resolver el Problema del Viajante (TSP). El algoritmo utiliza agentes inteligentes (hormigas) que depositan feromonas para encontrar rutas óptimas en grafos de ciudades.

## ⚙️ Parámetros del Modelo simple
* **Evaporación (rho):** 0.1
* **Constante de depósito (Q):** 10.0
* **Entrada:** Matrices de adyacencia con distancias entre nodos.

## 🛠️ Ejecución
El código incluye un manual de instrucciones (`docs/soluciones.txt`) y 5 ejemplos distintos con sus soluciones esperadas en la carpeta `data/`. Se ejecuta llamando a la función `procesa` e indicando el archivo de datos y el número de iteraciones que se quieren realizar.
